# Airbnb Clone

A modern full-stack Airbnb-inspired web application built using Node.js, Express.js, MongoDB, Mongoose, EJS, Tailwind CSS, and Multer.

---

# Features

- User Authentication (Signup/Login)
- Session Management
- Host & Guest Roles
- Add/Edit/Delete Properties
- Upload Property Images
- Add to Favourites
- Responsive Tailwind UI
- Dynamic Navbar
- MongoDB Database Integration

# Installation

## Clone Repository

```bash
git clone https://github.com/vaibhav-personal/airbnb-mongoose.git
````

## Open Project

```bash
cd airbnb-mongoose
```

## Install Dependencies

```bash
npm install
```

## Create .env File

```env
MONGODB_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
```

## Run Project

```bash
npm start
```

or

```bash
npm run dev
```
